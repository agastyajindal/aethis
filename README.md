# Aethis

**Intelligent Multivoicer + Guitar Processor in One Plugin**

The official documentation and showcase website for Aethis - a complete guitar processing solution combining professional amplifier modeling with a sophisticated 4-voice pitch shifter and harmonizer.

## Features

- **Three Amplifiers** - Noctis (Modern High-Gain), Rosaria (Vintage Chime), Lumina (Classic Rock)
- **4-Voice Multivoicer** - Pitch shifting, harmonization, scale quantization, and vocoder mode
- **Full Effects Chain** - Pre/Post EQ, Drive, Chorus, Delay
- **MIDI Control** - Real-time chord-following harmonies

## Website

Visit the live site: [agastyajindal.github.io/aethis](https://agastyajindal.github.io/aethis/)

```bash
# Install dependencies

# Start development server
npm run dev

# Build for production
npm run build
```
