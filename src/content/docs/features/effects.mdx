<BentoGrid>
  <FeatureCard title="Pre-Amp EQ" color="#EC4899">
    Shapes signal going **INTO** the amp:
    - Changes how the amp responds
    - Affects distortion character
    - Boosting mids = more mid-focused crunch
  </FeatureCard>
  <FeatureCard title="Post-Amp EQ" color="#94A3B8">
    Shapes signal coming **OUT OF** the amp:
    - Final polish and correction
    - Doesn't change amp character
    - Better for surgical adjustments
  </FeatureCard>
</BentoGrid>
