# Multivoicer

The heart of Aethis—a sophisticated **4-voice pitch shifter and harmonizer** that can create anything from subtle doubling effects to full vocal-style harmonies and otherworldly synth textures.

## Scale Quantization

Lock your harmonies to musical scales for always-in-key results.

<ControlCard
  title="Scale Settings"
  color="#FFD97B"
  controls={[
    { name: "Quantize Enabled", range: "On/Off", description: "Enable scale-aware harmonization" },
    { name: "Scale Type", range: "8 scales", description: "Chromatic, Major, Minor, Harmonic Minor, Pentatonic Major/Minor, Dorian, Mixolydian" },
    { name: "Root Key", range: "C to B", description: "Select the root note of your scale" }
  ]}
/>

When enabled, the harmonizer adjusts voice intervals to stay within the selected scale. This prevents "wrong" notes in your harmonies—your guitar turns into a choir of perfectly harmonized voices.

## MIDI Harmonizer Mode

<ControlCard
  title="MIDI Control"
  color="#FF8585"
  controls={[
    { name: "MIDI Mode", range: "On/Off", description: "Voice intervals follow incoming MIDI notes for real-time chord-following harmonies" }
  ]}
  columns={1}
/>

Connect a MIDI controller or DAW track, play chords via MIDI, and the Multivoicer will set voice intervals to match those notes in real-time. Your guitar provides the tone while MIDI controls the harmony.
